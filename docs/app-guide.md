# LogiCaX アプリ説明書

LogiCaX は、日常の計算式をカテゴリごとに保存し、必要な数値を入力してすぐに計算できるアプリです。

料理、旅行、健康、金融、学習などのプリセット式を使えるほか、自分で作った式を保存して繰り返し使えます。

## 基本的な使い方

1. カテゴリ一覧から、使いたい分類を開きます。
2. 計算したい式を選択します。
3. 計算画面で、必要な数値を入力します。
4. 入力に合わせて計算結果が更新されます。

よく使う式は、星印を押すと「よく使う式」に登録できます。

## カテゴリと検索

式はカテゴリごとに管理されます。カテゴリはフォルダとして表示されます。

検索欄では、次の内容を検索できます。

- 式名
- 式の内容
- コメント
- 分類名

カテゴリ名には、階層や並列分類を含めることができます。

```text
医療:腎臓/電解質
```

`:` は大分類と小分類の区切り、`/` は並列カテゴリの区切りとして使います。

## 式を使う

式を選ぶと、入力欄と計算結果が表示されます。

数値を入力すると、結果が自動で更新されます。ラジオ形式の入力がある式では、選択した項目の数値が計算式に使われます。

計算画面では、次の操作ができます。

- 数値の入力
- ラジオ項目の選択
- 入力値のリセット
- 式の編集
- 個別バックアップ
- フロート電卓の表示

## 式を作成する

カテゴリ一覧の右上にある新規作成ボタンから、「新規計算」を選びます。

式作成では、主に次の項目を設定します。

- 式名
- 分類名
- 入力変数
- 計算結果
- 計算式
- コメント
- リスク分類

式の中では、表示ラベルではなく変数名を使います。新しい入力変数には `v1`, `v2` のような名前が入り、新しい結果には `r1`, `r2` のような名前が入ります。

例:

```text
v1 * v2
```

「粉量」「年齢」などの表示ラベルは、画面に表示するための名前です。計算式の中では、対応する変数名を使ってください。

関数の書式は、式作成画面のメニューにある「関数ヘルプ」から確認できます。

## 使える関数の例

LogiCaX では、次のような関数を式に使えます。

| 種類         | 関数例                                                       |
| ------------ | ------------------------------------------------------------ |
| 条件         | `IF`, `IFERROR`, `IFS`, `BETWEEN`, `CLAMP`, `AND`, `OR`, `NOT` |
| 集計         | `SUM`, `SUMIF`, `AVERAGE`, `AVERAGEIF`, `MIN`, `MAX`, `MEDIAN` |
| 数学・丸め   | `ROUND`, `ROUNDUP`, `ROUNDDOWN`, `MROUND`, `TRUNC`, `ABS`, `SQRT`, `POWER` |
| 日付・時刻   | `TODAY`, `NOW`, `DATE`, `DAYS`, `WEEKDAY`, `YEAR`, `MONTH`, `DAY`, `DATEDIF` |
| 文字列・選択 | `LEN`, `LEFT`, `RIGHT`, `MID`, `CONCAT`, `TEXT`, `CHOOSE`    |
| 財務         | `PMT`, `PV`, `FV`, `NPV`                                     |

### 式の例

```text
IF(v1 >= 80, 1, 0)
```

```text
ROUND(v1 * v2, 1)
```

```text
SUM(v1, v2, v3)
```

## 式やカテゴリを整理する

右上のメニューから「移動」を選ぶと、式やフォルダを整理できます。

1. 移動したい式またはフォルダにチェックを入れます。
2. 移動先の行に表示される青い矢印アイコンを押します。
3. 移動が終わったら「完了」を押します。

式を長押しすると、次の操作を選べます。

- 星印を付ける / 外す
- 修正
- コピー
- この式をバックアップ
- 削除

## バックアップする

右上のメニューから「バックアップ」を選ぶと、式を JSON ファイルとして書き出せます。

バックアップ対象は次から選べます。

- 全ての式
- 選択した式
- フォルダごと
- 個別の式

書き出した JSON ファイルは、共有、保存、別端末への移行、コミュニティ投稿などに利用できます。

フォルダごとバックアップする場合は、選択したフォルダ内の式が1ファイルずつ書き出されます。

## バックアップを読み込む

右上のメニューから「バックアップ読込」を選ぶと、保存済みの JSON ファイルを読み込めます。

「URL から読込」では、共有されたバックアップ JSON の URL を入力して読み込めます。

同じタイトルの式がある場合は、新しい式で置き換えるか確認されます。置き換えると、既存の同名式は削除され、読み込んだ式が追加されます。

## コミュニティ式を追加する

右上のメニューから「コミュニティ式」を開くと、公開されている式パックを一覧から追加できます。

コミュニティ式はユーザー提供のサンプルです。内容を確認したうえで利用してください。

## 設定

設定画面では、次の項目を変更できます。

- 表示言語
- アプリ内の文字サイズ
- カテゴリ一覧の並び方
- バックアップフォルダ
- 免責事項の確認
- 設定のリセット
- 工場出荷状態への復元

工場出荷状態に戻すと、作成・編集・並べ替えした式と分類は削除され、アプリ内の式はプリセット状態に戻ります。必要な式は先にバックアップしてください。

## コミュニティ投稿

自作の式パックを共有したい場合は、LogiCaX のバックアップ機能で JSON ファイルを書き出し、コミュニティ用リポジトリの投稿フォームから送信できます。

投稿する JSON ファイルには、個人情報、秘密情報、API キー、第三者の権利を侵害する内容を含めないでください。

## 注意事項

LogiCaX の計算結果は、入力内容と登録された式に基づく参考値です。

医療、健康、金融、法律など重要な判断には、LogiCaX の結果だけを使用しないでください。必要に応じて専門家や公的情報を確認してください。

コミュニティ式や共有されたバックアップを読み込む場合も、式の内容と出どころを確認してから使用してください。
