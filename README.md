# LogiCaX Community Pack

LogiCaX アプリ用のコミュニティ式パック配信リポジトリです。

## このリポジトリは何？

[LogiCaX](https://apps.apple.com/) アプリ内の「コミュニティ式」から、ここに登録された計算式パックをワンタップで取り込めます。

## 投稿方法

[**📝 式パック投稿フォームを開く**](https://github.com/nabe-git1213/logicax-community/issues/new?template=pack_submission.md)

### 手順
1. LogiCaX アプリで自作の式を作成
2. 「バックアップ」機能で JSON ファイルにエクスポート
3. 上記の **投稿フォーム** を開く
4. フォームの項目を埋め、エクスポートした JSON ファイルを **ドラッグ&ドロップ**で添付
5. **Submit new issue** を押して投稿完了

メンテナーがレビュー後、`packs/` に取り込み・`gallery.json` に追加します。
（採用までに数日かかる場合があります）
